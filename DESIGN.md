# NSDraw

NSDraw is staticly hosted web project that enables visitors to draw using code! It features an Ace code editor, a menu with some example programs, and a play button which runs the Javascript canvas application in a full page.

# library

NSDraw.js contains some convenience functions like `draw_circle` which takes an object of parameters and produces the corresponding canvas ctx call.

The library is imported after the user's program.js.
